# Resume-Screening-using-Machine-Learning
This project focuses on building an end-to-end Resume Screening System using Machine Learning and Natural Language Processing (NLP) techniques. The goal is to automate the process of analyzing resumes and predicting their job categories, similar to real-world CV screening systems used in HR and recruitment platforms.
